# SS仕様書

## システム方式

### バックエンド

Supabaseを利用。
無料枠利用はPostgres 500MB
ストレージ50MB

### フロントエンド

Next.js + Typescriptを利用。
UIのフレームワークとしてMUI v5を利用。
React Testing Frameworkでテストを行う。

### データベース

```mermaid
erDiagram
    User ||--o{ GroupUser : ""

    User {
      UUID user_id PK

      timestamp updated_at
    }
    Group {
      UUID id PK
      string name

      string name
      string description
      string image_key
      string read_by
      string created_by

      timestamp created_at
      timestamp updated_at
    }
```
